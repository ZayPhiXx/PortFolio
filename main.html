Mes projets

Jeu de Dactylographie • 2021

Ayant pris NSI en première au lycée, j'ai dû réaliser un projet de groupe en fin d'année. Mon équipe et moi nous sommes orientés vers un jeu de dactylographie, car nous voulions un projet ludique et utile. La dactylographie est le fait de taper rapidement sur un clavier d'ordinateur sans regarder les touches. Cette compétence est très utile pour les métiers de l'informatique, et nous avons donc décidé de créer un jeu pour l'apprendre.

Pour réaliser ce projet, j'ai utilisé le langage Python, que ce soit pour l'interface graphique ou le moteur du jeu. J'ai aussi utilisé le module Pygame pour la gestion des événements et des images. Ce projet m'a permis de découvrir le monde du jeu vidéo et de m'améliorer en Python.

Python


Plugin pour un serveur Minecraft • 2023

J'ai réalisé ce projet car je rencontrais un problème. En effet, avec des amis, je jouais au Loup Garou sur un serveur Minecraft. Or, nous avions envie de jouer sur notre propre serveur. Le plugin pour jouer au Loup Garou n'était pas complet. En effet, il manquait un menu pour choisir les rôles et les paramètres de la partie. J'ai donc décidé de créer un plugin pour répondre à ce besoin.

Ce projet m'a pris une semaine de développement, mais grâce à ce projet, j'ai pu apprendre les bases du langage Java, ce qui m'a d'ailleurs aidé en première année de BUT Informatique.

Pour réaliser ce projet, j'ai utilisé le langage Java et le logiciel IntelliJ IDEA pour le développement du plugin.

Java


Site Web Orange Business • 2024

Dans le cadre de ma première année de BUT Informatique, j'ai dû réaliser en groupe un site web pour Orange Business, destiné à des élèves de 3ème qui veulent faire leur stage. Nous avions des contraintes à respecter, comme par exemple que le site soit écoresponsable, donc le plus simple et épuré possible. Nous avons dû réaliser un cahier des charges pour définir les besoins du client, puis nous avons réalisé la maquette du site pour que le client puisse voir à quoi ressemblerait le site. Ensuite, nous avons rassemblé toutes les informations nécessaires sur Orange Business pour alimenter le contenu du site. Enfin, nous avons commencé la réalisation du site.

Pour réaliser ce projet, nous avons utilisé les langages HTML et CSS. Nous avons aussi utilisé Git pour partager et travailler sur le même code.

HTML CSS


Jeu Faerun • 2024

Lors de ma première année de BUT Informatique, j'ai eu l'occasion de créer un jeu sans interface graphique. Le jeu s'appelle Faerun. Le but du jeu est simple : nous avons un plateau composé de cases avec deux châteaux de deux couleurs différentes (bleu et rouge) aux deux extrémités du plateau. Les châteaux contiennent des guerriers qui avancent à tour de rôle sur les cases. Dès que des guerriers bleus rencontrent des guerriers rouges, ils se battent. Le jeu se termine lorsqu'un des deux châteaux n'a plus de guerriers.

Pour réaliser ce projet, j'ai utilisé le langage Java, ce qui m'a permis de grandement m'améliorer avec ce langage.

Java
